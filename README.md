# Optimal experiment design

This is the repository for code and demos for the paper *Optimal experiment design for practical parameter identifiability and model discrimination* by Liu, Baker, Maini, 2025, available on arXiv at [arxiv.org/abs/2506.11311](https://arxiv.org/abs/2506.11311).

See profile_likelihood_control.m for steps leading up to Fig.1.

See minimize_conf_int_range.m for steps leading up to Fig.2, 4. 

See sensitivity.m for steps leading up to Fig.3.

See twomodel_control.m for steps leading up to Fig.5, which is similar for the steps for Fig.6-11.
